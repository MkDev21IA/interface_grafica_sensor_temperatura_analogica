Detailed Description

Classe principal da interface gráfica.

Herda de QMainWindow e compõe todos os widgets da UI, inicia o listener UDP e atualiza a interface com os dados recebidos.

main_window.MainWindow.update_data ( self,

Slot: O "coração" da UI.

Chamado quando o listener UDP emite novos dados.
