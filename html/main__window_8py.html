Detailed Description

Implementação da janela principal (Dashboard) da interface gráfica.

Esta classe é o coração da UI. Ela usa PyQt6 e pyqtgraph para criar um dashboard que exibe dados de sensores em tempo real, incluindo valor atual, gráfico histórico, alertas e opções de log.
