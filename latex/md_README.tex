Sistema embarcado para {\bfseries{monitoramento remoto de condições térmicas}}, desenvolvido com o {\bfseries{kit STM32\+MP1-\/\+DK1}} e o sensor analógico {\bfseries{KY-\/013}}, com transmissão contínua via {\bfseries{UDP}} e supervisão em {\bfseries{interface gráfica multiplataforma}}.

\DoxyHorRuler{0}
\hypertarget{md_README_autotoc_md2}{}\doxysection{\texorpdfstring{🚀 Visão Geral}{🚀 Visão Geral}}\label{md_README_autotoc_md2}
Cargas sensíveis exigem {\bfseries{rastreabilidade e monitoramento constante}}. Este projeto implementa um sistema embarcado que\+:
\begin{DoxyItemize}
\item Lê a temperatura em tempo real pelo {\bfseries{ADC do STM32\+MP1-\/\+DK1}} ~\newline

\item Converte o valor bruto para {\bfseries{graus Celsius}} ~\newline

\item Transmite os dados via {\bfseries{UDP}} para um servidor na rede local ~\newline

\item Exibe as leituras em uma {\bfseries{interface gráfica no PC}}, com histórico e alertas configuráveis ~\newline

\end{DoxyItemize}

O sistema segue uma arquitetura {\bfseries{IoT modular}}, permitindo integração futura com outros sensores (umidade, luminosidade, vibração etc.).
